# modern-welcome-housing

Create a landing page (LP) for a housing sales company featuring a white background with yellow and black as accent colors. The design should be modern and welcoming, with an emphasis on cleanliness and ease of navigation. Include sections for featured properties, customer testimonials, and contact information. Ensure the layout is responsive for both desktop and mobile devices.
